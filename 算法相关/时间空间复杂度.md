# 时间空间复杂度

时间复杂度
递归主定理

| 类型 | 主定理 | 时间复杂度 |
| --- | --- | --: |
| 二分法 | T(n)=T(n/2)+O(1) | O(logn) |
| 二叉搜索树 | T(n)=2T(n/2)+O(1) | O(n) |
| 二维矩阵 | T(n)=2T(n/2)+O(logn) | O(n) |
| 归并排序 | T(n)=2T(n/2)+O(n) | O(nlogn) |

    
空间复杂度

两个关键条件：1.数组长度        一维数组长度为n空间复杂度为O(n)
            2.递归深度   递归深度的最大值就是空间复杂度的最大值
