# 搭建开发环境


## 1.基础环境搭建

- 安装Node.js
    - Node.js version >= 12
    - npm config set registry https://registry.npm.taobao.org
- 安装Yarn
    - npm install -g yarn

- 安装React Native 脚手架
    - npm install -g react-native-cli

ps:windows下只能搭建Android开发环境
